##### 拷贝构造函数

* 功能：用一个对象初始化另一个对象

  * ```c++
    Test t(10);
    // 调用拷贝构造
    Test t2 = t;
    // 等价于下面
    Test t2(t);
    ```

* 声明：只有一个参数并且参数为该类对象的引用

  * `Test(const Test& other);`
  * 这里必须是`&`引用，不然使用值传递，又要调用拷贝构造 造成递归调用

* 若类中没有说明拷贝构造函数，则系统自动生成一个缺省复制构造函数，作为该类的公有成员

##### 拷贝构造函数调用的几种情况

* 当**函数形参**是**类对象**时，调用函数时，进行形参与实参结合使用
  * 在内存建立一个**局部对象**，将实参**拷贝到新的对象**中。调用**拷贝构造函数**
