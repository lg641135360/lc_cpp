##### 给定一个整数 `n` ，返回 `n!` 结果中尾随零的数量。

* 连续的零

* ![image-20210926212258930](%E9%98%B6%E4%B9%98%E5%90%8E%E7%9A%84%E9%9B%B6.assets/image-20210926212258930.png)

* 思考

  * 为什么会出现0？阶乘？
    * 一个数乘以10才会有一个尾数0出现
  * 找出阶乘的表达式中出现多少个10
    * 10 = 2*5，5出现过程中，2很多
    * 只用考虑5的个数
  * 什么数字会贡献5
    * 5的倍数贡献一个0，25的倍数再贡献一个
  * n/5 + n/25 + n/125 + ...

* 代码

  * ```c++
    int trailingZeroes(int n) {
        int count = 0;
        // 这里找到所有是5倍数的i
        for(long long i=5;n/i;i*=5)
        	count+=n/i;
        return count;       
    }
    ```
